# DenLifors - Интернет-магазин БАД

Современный интернет-магазин биологически активных добавок с админ-панелью.

## Возможности

### Пользовательская часть
- Главная страница с баннерами и товарами
- Каталог товаров с фильтрацией и поиском
- Страница товара с подробным описанием
- Корзина покупок
- Оформление заказа
- Личный кабинет пользователя
- Партнёрская программа
- FAQ и документация

### Админ-панель
- Управление товарами (добавление, редактирование, удаление)
- Управление категориями
- Управление заказами
- Управление баннерами
- Просмотр статистики
- Управление пользователями

## Установка

### Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Apache/Nginx веб-сервер
- XAMPP (рекомендуется для разработки)

### Шаги установки

1. **Клонирование/копирование проекта**
   ```bash
   # Скопируйте файлы в папку htdocs вашего XAMPP
   # Путь: C:\xampp\htdocs\DenLiFors
   ```

2. **Настройка базы данных**
   
   **Вариант 1 (Автоматический):**
   - Откройте в браузере: http://localhost/DenLiFors/install.php
   - Скрипт автоматически создаст базу данных и все таблицы
   
   **Вариант 2 (Вручную):**
   - Откройте phpMyAdmin (http://localhost/phpmyadmin)
   - Импортируйте файл `database/schema.sql`
   - Или выполните SQL-запросы вручную

3. **Настройка конфигурации**
   - Откройте `config/database.php`
   - Проверьте настройки подключения к БД (по умолчанию для XAMPP):
     ```php
     define('DB_HOST', 'localhost');
     define('DB_USER', 'root');
     define('DB_PASS', '');
     define('DB_NAME', 'DenLiFors');
     ```

4. **Создание папок для загрузок**
   ```bash
   mkdir uploads
   mkdir uploads/products
   chmod 755 uploads
   chmod 755 uploads/products
   ```

5. **Доступ к сайту**
   - Откройте браузер: http://localhost/DenLiFors
   - Админ-панель: http://localhost/DenLiFors/admin/

### Учетные данные по умолчанию

**Администратор:**
- Email: `admin@denlifors.ru`
- Пароль: `admin123`

⚠️ **ВНИМАНИЕ:** Измените пароль администратора после первой установки!

## Функциональность

### Товары
- Полная информация о товаре (описание, состав, способ применения)
- Изображения товаров
- Цены и скидки
- Остатки на складе
- Категории

### Корзина
- Добавление товаров в корзину
- Изменение количества
- Удаление товаров
- Сохранение корзины для авторизованных и неавторизованных пользователей

### Заказы
- Оформление заказа
- Отслеживание статуса заказа
- История заказов в личном кабинете

### Админ-панель
- Полное управление товарами
- Управление заказами (изменение статусов)
- Управление категориями
- Просмотр статистики
- Управление баннерами

## Разработка

### Добавление нового товара через админ-панель
1. Войдите в админ-панель
2. Перейдите в "Товары" → "Добавить товар"
3. Заполните форму
4. Сохраните

### Добавление изображений
В текущей версии загрузка изображений через интерфейс не реализована. 
Изображения необходимо загружать вручную в папку `uploads/products/` 
и указывать имя файла в базе данных.

4. Версию PHP (должна быть 7.4+)
