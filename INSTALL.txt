==========================================
  DENLIFORS - УСТАНОВКА И НАСТРОЙКА
==========================================

БЫСТРАЯ УСТАНОВКА:

1. Скопируйте все файлы в папку: C:\xampp\htdocs\DenLiFors

2. Убедитесь, что XAMPP запущен (Apache и MySQL)

3. Откройте в браузере: http://localhost/DenLiFors/install.php
   Это автоматически создаст базу данных и таблицы

4. Войдите в админ-панель:
   URL: http://localhost/DenLiFors/admin/
   Email: admin@denlifors.ru
   Пароль: admin123

5. ОБЯЗАТЕЛЬНО измените пароль администратора после первого входа!


НАСТРОЙКА:

- База данных настраивается в файле: config/database.php
- По умолчанию для XAMPP:
  * Хост: localhost
  * Пользователь: root
  * Пароль: (пусто)
  * База данных: DenLiFors


СТРУКТУРА ПРОЕКТА:

- index.php - главная страница
- catalog.php - каталог товаров
- product.php - страница товара
- cart.php - корзина
- dashboard.php - личный кабинет (дашборд)
- admin/ - админ-панель


ВАЖНО:

1. Для загрузки изображений товаров создайте папку uploads/products/
   (уже создана при установке)

2. Загрузка изображений через интерфейс пока не реализована.
   Загружайте изображения вручную в папку uploads/products/
   и указывайте имя файла при редактировании товара.

3. Для работы сайта необходимо, чтобы Apache и MySQL были запущены.


ПОДДЕРЖКА:

При возникновении проблем проверьте:
- Запущен ли Apache и MySQL в XAMPP
- Правильность настроек БД в config/database.php
- Существует ли база данных DenLiFors
- Права доступа к папке uploads
