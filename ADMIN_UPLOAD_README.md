# Инструкция по использованию загрузки изображений в админ-панели

## Установка дефолтного изображения для товаров

1. Загрузите изображение бутылки в папку `uploads/products/`
2. Переименуйте его в `product-default.jpg`
3. Перейдите в админ-панели на страницу `admin/set-default-product-image.php`
4. Нажмите кнопку "Установить изображение для всех товаров"

## Загрузка изображений для товаров

### Главное изображение
- Перетащите изображение в область "Главное изображение" или нажмите для выбора файла
- Поддерживаемые форматы: JPG, PNG, GIF, WebP
- Максимальный размер: 5MB

### Галерея изображений
- Перетащите несколько изображений в область "Галерея изображений" или нажмите для выбора файлов
- Можно выбрать несколько файлов одновременно
- Чтобы удалить изображение из галереи, нажмите кнопку "×" на изображении

## Загрузка изображений для баннеров

1. Перейдите в раздел "Баннеры" в админ-панели
2. При создании или редактировании баннера перетащите изображение в область загрузки
3. Изображение будет автоматически загружено и сохранено

## Технические детали

- Все изображения сохраняются в папке `uploads/products/` (для товаров) и `uploads/banners/` (для баннеров)
- Имена файлов генерируются автоматически для предотвращения конфликтов
- Изображения товаров сохраняются в базе данных:
  - `image` - главное изображение (имя файла)
  - `images` - галерея изображений (JSON массив имен файлов)
